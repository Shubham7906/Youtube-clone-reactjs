Built this Youtube Clone Single Page Application using ReactJs, TailwindCSS and the RapidAPI as the data source.

# Features
- Video Card
- Category Filters
- Search Functionality
- Related Videos Functionality
- Video Player
- Responsive Design

# Tech Stack
- Reactjs
- TailwindCSS
- RapidAPI
- React-router
